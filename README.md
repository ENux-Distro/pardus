# Pardus Package Manager

In order to install the Pardus Package Manager on your Pardus system, you need to these steps:

1- Git clone the repo via `sudo apt install git && git clone --depth 1 https://github.com/ENux-Distro/pardus.git`

2- Change your current directory to the repo's directory via `cd /path/to/pardus`

3- Run the setup script via `bash setup`

After everything went succesfully, you now have installed the Pardus Package Manager to Pardus
