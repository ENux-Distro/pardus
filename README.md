# Pardus-Installer

In order to install the Pardus-Installer on your **live** Pardus system, you need to these steps:

1- Git clone the repo via `sudo apt install git && git clone --depth 1 https://github.com/ENux-Distro/Pardus-Installer.git`
2- Change your currenty directory to the repo's directory via `cd Pardus-Installer`
3- Run the setup script via `bash setup`

After everything went succesfully, you now have installed the Pardus-Installer for Pardus Live
